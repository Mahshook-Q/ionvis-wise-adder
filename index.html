WISE Architecture

Wiring Using Integrated Switching Electronics — the solution to the wiring challenge of scaling trapped-ion quantum computers to 1000+ qubits.

cable

The Wiring Challenge

A 1000-qubit trapped-ion chip requires ~10 electrodes per qubit, and conventional control needs one dedicated DAC per electrode — that's ~10,000 control lines. This is beyond the physical limit of chip I/O and makes scaling physically impossible with standard approaches. The arithmetic units (adders) that underpin quantum advantage hit this wall first during carry propagation.

Impact on Quantum Addition

Without WISE (Ripple-Carry)

Each carry qubit needs dedicated electrode DAC. Sequential gates require sequential DAC switching — adds latency beyond T₂ coherence time. Carry chain decoheres before completion.

With WISE (Transport-Assisted Gates)

Ions shuttled to shared gate zones. Only ~100 dynamic DACs needed regardless of chain length. Sample-and-hold eliminates switching noise during gates. Carry propagation stays coherent.

WISE Gate Speed

40–2600 gate layers/second achievable on 1000-qubit device. Adder circuits that previously failed due to depth limits now execute within coherence window.

State Preparation

Ions are Doppler-cooled and sideband-cooled to the motional ground state n̄ ≈ 0.04. The WISE switch network pre-charges all shim electrodes before cooling begins, then disconnects — the sample-and-hold capacitors maintain electrode potentials throughout the cooling sequence without any switching noise.

T_cool ~ 2ms Shim DACs: pre-charged Switch: OPEN (no noise)

dangerous

Cascading Noise Failure

Ripple-Carry algorithm: carry propagation requires 5+ sequential 2-qubit gates. Each MS-gate introduces ~0.5% error and ~1ms of coherence time. The total gate stack exceeds T₂ ≈ 4.2ms. Electrode switching noise from carry qubit DACs leaks into neighboring ion potential wells, causing decoherence cascade. This is the exact failure mode WISE architecture addresses by separating switching operations from gate operations via sample-and-hold capacitors.

T₂ budget: 4.2ms Gate stack: ~5ms Crosstalk: 15% WISE fix: sample-and-hold

Ion (The Dot)

Visual Element 02

Excitation Field

Expanding color-coded circle around an ion. Radius ∝ Rabi frequency. When a 729nm laser pulse resonates with the transition frequency ΔE=hν, the field appears. Pink = |e⟩ electronic state.

ΔE = hν |e⟩ STATE 729nm laser

Rydberg Dressing (Ring Overlap)

When excitation fields of two ions overlap, Rydberg-blockade creates a conditional interaction — the basis of the MS gate. Larger field = stronger Rabi coupling. Overlap = entanglement active. Implemented via off-resonant dressing laser.

Visual Element 04

WISE Transport

Green dashed arrow = ion shuttle. In WISE architecture, rather than routing signals to where qubits sit, the qubits (ions) are physically transported to shared gate zones. ~100 DACs handle this via integrated switch network regardless of total qubit count.

QCCD architecture 100μs shuttle time
Visual Element 05

Noise & Failure

Red ion (⚠) = decoherence or error state |?⟩. Flashing red wires = crosstalk from neighboring electrode potentials. Cascading: when carry qubit T₂ budget exhausted, qubit enters mixed state — no longer usable for computation.

T₂ ≈ 4.2ms |?⟩ mixed state

Photon Readout

370nm fluorescence laser drives cycling transition. Excited ions emit photons (shown as rising dots). Ground state ions remain dark. PMT/CCD collects photons → classical bit determination. Each photon burst = one qubit measurement.

Reference Paper

This simulator incorporates the WISE architecture from: Malinowski, M., Allcock, D.T.C. & Ballance, C.J. — "How to wire a 1000-qubit trapped-ion quantum computer", PRX Quantum 4, 040313 (2023), arXiv:2305.12773. The WISE-Optimized algorithm in this simulator demonstrates how transport-assisted gates and sample-and-hold electrode control enable quantum arithmetic circuits that would otherwise fail due to T₂ decoherence.
