# Practical Implementation & Feasibility

**What works today:**
- 2-qubit adders run on IonQ Harmony, Quantinuum H1 with ~99.5% 2Q fidelity
- Shuttling demonstrated with 1e-4 error

**What WISE enables:**
- Reduces cryogenic I/O, the current bottleneck
- Keeps error budget same (switches add <1e-5 error)

**IonVis shows:**
1. Ideal simulation
2. WISE-sim with multiplexing delay
3. Real hardware counts (overlay)

This demonstrates feasibility gap closing.
